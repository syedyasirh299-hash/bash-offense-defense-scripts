# Bash Offense & Defense Scripts

A curated collection of Bash scripts focused on **offensive security**, **defensive monitoring**, and **Linux automation**.  
This repository demonstrates practical command-line skills used in cybersecurity labs and real-world scenarios.

---

## Key Highlights
- **Network Scanning** – Ping and port scanning scripts
- **Directory Brute Force** – Simple web path discovery tool
- **System Monitoring** – Disk alerts and process watchdog
- **Password Strength Check** – Basic password validation
- **Backup Automation** – Simple backup script
- **Log Analysis** – Top IP extraction utility

---

## Scripts Overview
| Script | Purpose |
|--------|---------|
| `01_ping_scanner.sh` | Scan multiple hosts using ping |
| `02_port_scanner.sh` | Check open TCP ports |
| `03_user_checker.sh` | Verify user existence |
| `04_backup_script.sh` | Create simple backups |
| `05_disk_alert.sh` | Disk usage warning script |
| `06_directory_bruteforce.sh` | Discover web directories |
| `07_csv_age_filter.sh` | Filter CSV data by age |
| `08_password_strength.sh` | Basic password strength check |
| `09_top_ips.sh` | Show most active IPs from logs |
| `10_process_watchdog.sh` | Monitor and restart processes |
